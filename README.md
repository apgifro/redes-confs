# Serviços de Redes

## Sobre

Configurações para prova de Serviços de Redes I.

Ferramentas e serviços:

- VirtualBox `7.0` + Debian `11 Bullseye`
- DHCP `isc-dhcp-server` (dns1)
- DNS `bind9` (dns1 e dns2)
- Apache `apache2` (web)
- FTP + Filezilla e NFS (storage)
